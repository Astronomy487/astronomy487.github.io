use the following little script

<script src="https://astronomy487.github.io/averagefont/averagefont.js"></script>

any element marked with the attribute 'averagefont' font will be converted. the content of that attribute is a comma-separated list of font-families (defaults to Arial, Verdana, Tahoma, Trebuchet MS). a font can also include a font-weight after a slash (i.e. averagefont="Inter/bold")

elements with the 'averagefont' attribute can also include bgcolor and txtcolor to specify background a text color, but NOTE that things get CLAMPED in funky ways!!! this is because the essence of the trick is applying filter: contrast(500); !
