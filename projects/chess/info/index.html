What is this?

This is a game of chess I've been working on to practice JavaScript. It doesn't have full functionality yet: you can't yet promote, castle, or en passant. There is no check/checkmate detection for the current board state or for moves (trust me I tried really hard), so the objective is to actually capture the opponent's king. If you're looking for a fuller chess experience right now, go to chess.com or something.

Instead, my version of chess designed to be customizable and allow for alternative chess modes with alternative pieces or board arrangements.

How do I make a gamemode?

I intend to make a proper editor in the future, but for now you have to write a .js file that contains the gamemode's pieces, starting board, and list of teams (as of right now, you can only rotate teams 180°). If you want a .js file to model yours after, use rules-standard.js.

To then play a gamemode you wrote, enter its URL after a question mark, like https://astronomy487.github.io/projects/chess?rules-960.js.
