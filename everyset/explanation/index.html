Every set ever - explanation

Motivation

The Zermelo-Fraenkel axioms of set theory only assume the existence of the empty set, and for the existence of sets containing other such sets. These are "pure" sets, which are just nestings in nestings (all the way down!) until the empty set.

You can make silly set-theoretic definitions of integers, ordered pairs, functions, and other useful things by declaring that some pure sets correspond to them.

The first few integers according to the Von Neuman construction for ordinals

  • $0=\emptyset$ (0)
  • $1={\emptyset}$ (1)
  • $2={\emptyset,{\emptyset}}$ (3)
  • $3={\emptyset,{\emptyset},{\emptyset,{\emptyset}}}$ (11)
  • $n+1=n\cup {n}$

An ordered pair, per Kuratowski's definition

  • $(a,b)={{a},{a, b}}$
  • $(0,1)={{\emptyset},{\emptyset, {\emptyset}}}$ (10)

The boolean set, which also equals the Von Neuman ordinal 2

  • ${0,1}={\emptyset,{\emptyset}}$ (3)

The cartesian self-product of the boolean set

  • ${0,1}^2={{{\emptyset}}, {{\emptyset},{\emptyset,{\emptyset}}}, {{{\emptyset}},{\emptyset,{\emptyset}}}, {{{\emptyset}}}}$ (5140)

A subset of this cartesian product, corresponding to boolean not function

  • ${{{\emptyset},{\emptyset,{\emptyset}}}, {{{\emptyset}},{\emptyset,{\emptyset}}}}$ (5120)

I find the notion comical that, whenever you use the number 3, you're secretly referring to this pure set. That's hilarious to me. It's funny to think that every function in calculus is some infinitely-complex set-full-of-sets, and that's what's _really_ going on.

Thus I thought it useful to list all of them.

Scheme

The correspondance between indices and sets relies on counting in binary. The $i$th set ($s_i$), where $i$ has the binary representation $\cdots b_4b_3b_2b_1b_0$, has as an element $s_0$ iff $b_0=1$, has $s_1$ iff $b_1=1$, and so on.

$$s_n\in s_i\leftrightarrow b_n=1$$

This makes it easy to find the elements of the $i$th set. To work backwards from a set and find its index,

  • Consider all elements of the set
  • Find their indices
  • For each (unique) index found, add $2^n$ to a running total

Set operations can be represented as bitwise arithmetic with their indices, if you really feel like it:

$${s_i}=s_{2^i}$$ $$s_i\cap s_j=s_{i\mathtt \& j}$$ $$s_i\cup s_j=s_{i\mathtt | j}$$ $${s_i,s_j,s_k,\cdots}=s_{2^i+2^j+2^k}\;(i\neq j\neq k\neq i)$$

The successor function for Von Neuman ordinals can be turned into index arithmetic: $$S(s_i)=2^i+i$$ This follows the sequence $0, 1, 3, 11, 2059, 6.619\times 10^{619}, \cdots$ (OEIS A034797).

Sets' complements doesn't really make sense here, because

  • A universal set would generate Russell's paradox here
  • Under the rules so far, $s_i'=U\setminus s_i$ would be $s_{-n-1}$, a negative-indexed set (using two's complement)
  • Negative-indexed sets are incompatible with using indices for the binary strings $\cdots b_4b_3b_2b_1b_0$

The cardinality of sets follows OEIS A000120, the number of 1s in the binary expansion. The rank of $s_i$ (set nesting depth) is the smallest number $R$ such that $2\uparrow\uparrow R>i$. The number of leaves on a set's tree (number of $\emptyset$s contained within) follows OEIS A033922.

My computer starts to slow down around index $10^{2000}$, somewhere in rank 5.

Weird things

  • The long-run proportion of sets containing $\emptyset$ is $\frac12$ (equal to the long-run proportion of odd numbers in $\mathbb N$) (according to this scheme)
    • A different mapping from $\mathbb N$ to pure sets could make any arbitrary proportion, including $0$ and $1$
    • This holds true for every set! Every set $s_i$ is contained in half of all sets! In the words of Pastor Jeff, "Worst-case scenario, there's a 50-50 chance, and I like those odds." (Young Sheldon S1E3)
